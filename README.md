# Todo List Web App

A simple Todo List web application built with **HTML, CSS, and vanilla JavaScript**.  
Users can add tasks, mark them as completed with a checkbox, and delete them from the list. Tasks are stored in the browser using **localStorage**, so they remain saved even after refreshing the page.

This project was created as a beginner frontend exercise to practice core JavaScript concepts such as **DOM manipulation, event handling, and dynamic element creation**.

---

## Features

- Add new tasks
- Mark tasks as completed using checkboxes
- Delete tasks
- Tasks persist using **localStorage**
- Simple and clean user interface

---

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **localStorage API**

---

## How It Works

1. The user enters a task in the input field.
2. When the **Add** button is clicked, JavaScript creates a new list item dynamically.
3. Each task contains:
   - A checkbox
   - The task text
   - A delete button
4. Tasks are stored in **localStorage** so they remain after refreshing the page.
